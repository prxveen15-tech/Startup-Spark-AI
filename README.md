# 🚀 Startup Spark AI

Startup Spark AI is a modern full-stack web application that helps aspiring entrepreneurs explore startup ideas through an interactive AI-inspired interface. The project features a responsive user interface, engaging animations, and a Node.js backend built with Express.

---

## ✨ Features

- 🎨 Modern glassmorphism UI
- 📱 Fully responsive design
- 🌌 Animated particle background
- 🚀 Startup idea generator
- ⚡ Interactive animations and effects
- 🖥️ Node.js + Express backend
- 🔧 Clean project structure
- 📂 Version controlled with Git and GitHub

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/prxveen15-tech/Startup-Spark-AI.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the backend

```bash
node backend/server.js
```

### 4. Open the frontend

Open:

```text
frontend/index.html
```

in your browser.
